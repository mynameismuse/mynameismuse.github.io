开发使用

在实际的开发中，你只需要下载组件包，并在项目中引入，就可以直接仿照例子中的 DEMO 进行开发了。

下载的 UI 代码库和补丁库，大致内容如下：

  • woUI
    • Button
    • Tabs
    • …
    • woUI
  • lib
    • es5-shim.min.js
    • es5-sham.min.js
    • ie.min.js

其中 woUI 为组件目录，里面可以单独引用某一个组件（包含 js/css 文件），也可以一次性引入 woUI 全部组件集合。

lib 目录主要为补丁目录，针对低版本浏览器需要进行打补丁才可以实现效果。
